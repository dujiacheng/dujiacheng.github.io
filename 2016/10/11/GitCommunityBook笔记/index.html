Veröffentlicht am 2016-10-11

#git的初始化

##1.Git配置
$ git config –global user.name “dujiacheng”
$ git config –global user.email “cleverdove4@163.com”
$ cat ~/.gitconfig #配置之后,会在当前用户目录下生成一个配置文件
[user]
email = cleverdove4@163.com
name = dujiacheng
这样是全局设置, 如果要在某个项目中使用公司邮箱,使用以下命令也就是不加global
$ git config user.name “dujiacheng”
$ git config user.email “dujiacheng@heyang.mail”

#正常的工作流程
$ git status # 查看git仓库状态
$ git diff #查看工作区的变化(比如file1已经被add了,再次修改后 再运行git diff 可以检查出修改了什么)
$ git diff –cached #查看缓存区的变化

$ git commit -a -m "add 3 files"   # 参数-a表示file1再次修改后不需要再次add就可以被commit到本地仓库
